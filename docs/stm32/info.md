## Supported MCUs

The following STM32 microcontrollers are supported by the MCU MDT:

Cortex-M0:

| Part Number   | Flash | RAM  | Notes                                                |
| ------------- | ----- | ---- | ---------------------------------------------------- |
| STM32F030F2P6 | 4 KB  | 2 KB | Minimal memory, same core/peripherals                |
| STM32F030F4P6 | 16 KB | 4 KB | Your MCU                                             |
| STM32F030F6P6 | 32 KB | 4 KB | Slightly more Flash                                  |
| STM32F030F8P6 | 64 KB | 8 KB | More memory, same registers                          |
| STM32F030C6P6 | 32 KB | 4 KB | Some pin differences, but core/peripherals identical |
